Sharpsell is a sales playbook automation platform.

- We define the Sales Playbook as an interactive guide which is repeatable and enforceable at scale, to make the sales person exhibit winning behaviour.
- It operates like a co-pilot, a teammate for the seller that adapts and serves them every step of the sales process. Helping them hit targets and achieve the goal.
- A Sales Playbook does this by providing learning, content, tools and answers that are required to close the sale.


Start integrating our SDK on your platform easily by following the documentation. We have attached platform-specific details, a reference project and an integration tutorial video to make the integration smoother.

## Pre-Requisites

### 1. Platform versions
<details>
<summary>Version details</summary>
<Tabs>
<TabItem value="Android">

1. The application should be migrated to AndroidX. Check if the following line is present in the project level `gradle.properties`. If this line is not present, the project needs to be migrated to [androidX](https://developer.android.com/jetpack/androidx/migrate)

```gradle
android.useAndroidX=true
```
2. The minimum android SDK version should be at least 21.
```gradle
minSdkVersion 21
```
:::info
To know more about android version and policy details [click here](/android_version_details)
:::
3. Firebase should be enabled and the `google-seriveces.json` file should be properly set up.

:::tip Firebase Setup
Please refer to the [Firebase Setup](/#2-firebase-setup) section to setup firebase for the required platform.
:::
